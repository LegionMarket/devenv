==========================
LegionMarket 1.4.1 release notes
==========================

*July 30, 2012*

This is the first security release in the LegionMarket 1.4 series, fixing several
security issues in LegionMarket 1.4. LegionMarket 1.4.1 is a recommended upgrade for
all users of LegionMarket 1.4.

For a full list of issues addressed in this release, see the `security
advisory`_.

.. _security advisory: https://www.djangoproject.com/weblog/2012/jul/30/security-releases-issued/
